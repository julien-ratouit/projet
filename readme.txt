Documentation :

	Si vous souhaitez acceder Ã  la documentation de notre programme, il vous faut aller dans le dossier "Documentation/html/index.html".

Compilation :
	
	Pour compiler notre programme il faut vous rendre dans "src" ensuite effectuez la commande 'make'.

Execution :

	Pour executer notre programme il faut vous rendre dans la source (emplacement de ce fichier text), puis effectuer la commande './bin/survive_students.exe'.
