fonction qui affiche des textures à l'ecran.

Dans cette fonction nous affichons les textures qui sont toujours utile, à chaque fois que le joueur clique sur un menu (le menu info ou le menu des actions par exemple) il faut tout re-afficher, donc pour eviter la redondance dans le code on a opté pour ce système de fonction.

Paramètres
texture_profest une variable de type SDL_Texture, on l'utilise pour toute nos textures à afficher à l'ecran.

◆ lancement_salle_prof()

void lancement_salle_prof ( param_t *  parametre)

fonction qui gère la salle des profs.

Dans cette fonction nous regardons ou est-ce que le joueur clique, si il clique sur un ordinateur alors on lance la fonction menu_action, au contraire si il décide de lancer une nouvelle journée alors on relance le programme et pour finir si il décide de sauvegarder alors on sauvegarde et on revient à la page principal.

Paramètres
statut_messest une valeur de type int qui permet de savoir si oui ou non le message de début est ouvert ou pas, si oui alors sa valeur est 1 sinon 0.
statut_infoest une valeur de type int qui permet de savoir si oui ou non le message d'information est ouvert ou pas, si oui alors sa valeur est de 1 sinon 0.
