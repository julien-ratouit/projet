Description détaillée

Ce programme permet de gerer les actions possibles lors de la partie.

On utilise ce programme quand le joueur se trouve dans la salle des profs mais également en jeu pour permettre d'agir en fonction de l'action utilisé.

Modifie le statut d'une action.

Si le joueur clic sur une action la fonction se lance et regarde la somme d'argent du joueur. Si la somme en question est assez grande alors l'action change de statut est devient acheté, or, si la somme n'est pas assez grande rien ne se passe.

Renvoie
On return un int qui défini la somme d'argent restant au joueur.

◆ clic_choix_sdp()

int clic_choix_sdp ( )

Attend un clic du joueur.

Ici, la fonction va attendre un clic du joueur, elle va permettre de modifier l'emplacement des actions achetés par le joueur. Quand le joueur clic sur une action alors la fonction renvoie l'id de l'action allant de 1 à 4 (sa position dans le tableau) ou alors, si le joueur clic sur un emplacement alors la fonction renvoie le numéros de l'emplacement en valeur negative allant de -1 à -4.

Renvoie
On return soit le numero de l'emplacement, l'id de l'action ou une valeur indiquant que l'on quitte le menu.

Initialise les informations d'une action.

Ici, on va initialiser les informations pour chaque action en static pour ne pas qu'elle se retrouve modifer dans la fonction main.

Paramètres
idcorrespond à sont emplacement dans le tableau des actions
prixest un entier qui represente sont prix
statutest un boolean qui permet de savoir si l'action est acheté ou non
equipeest un entier qui prend comme valeur sa place dans le tableau action_equipe
cdest un entier qui represente le temps d'attente entre chaque utilisation de l'action en jeu
action_realiseest un pointeur sur fonction qui pointe sur une fonction qui agit sur le jeu (reduire la barre sonore par exemple)
img_action_Xsont des liens pour récupérer les images approprié pour l'appropriés pour l'affichage
Renvoie
On return un action_t qui est notre action initialisé.
